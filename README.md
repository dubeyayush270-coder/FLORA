# FLORA

FLORA is a custom game engine written in C++ and inspired by Hazel, developed as a learning project while following The Cherno's Game Engine series.

## Overview

The goal of FLORA is to explore the fundamentals of game engine development by implementing core engine systems from scratch, including application management, event handling, rendering, window abstraction, and more.

## Current Features

* Application framework
* Engine entry point
* Sandbox application for testing
* Dynamic library (DLL) setup
* Visual Studio build configuration
* Premake-based project generation

## Technologies

* C++
* Visual Studio 2022
* Premake5
* Windows Platform

## Project Structure

* **FLORA/** - Core engine source code
* **SandBox/** - Test application built on top of the engine

## Learning Goals

This project is primarily focused on understanding:

* Engine architecture
* Rendering pipelines
* Event systems
* Memory management
* Platform abstraction
* Modern C++ practices
